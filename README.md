# Spider-quizz

Um quiz interativo no terminal com 3 perguntas sobre o universo do Homem-Aranha.
Este projeto foi desenvolvido com JavaScript utilizando o módulo readline do Node.js para praticar lógica de programação e interação via terminal.

# Tecnologias Utilizadas
JavaScript
Node.js
Módulo readline


# Como Executar o Projeto

Baixe e instale pelo site oficial:
https://nodejs.org/

Clone o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git

Execute o quiz usando o código abaixo
node quiz.js



# Como Funciona

O usuário responde digitando a, b, c ou d.
As respostas são normalizadas utilizando .toLowerCase() para evitar erros com letras maiúsculas.
Ao final do quiz, o sistema exibe um feedback baseado na quantidade de acertos.
O projeto utiliza o módulo readline para criar uma interface de entrada e saída diretamente no terminal.
